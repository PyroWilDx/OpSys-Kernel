# OpSys-Kernel

[**OpSys-Kernel**](https://github.com/PyroWilDx/OpSys-Kernel/) is an Operating-System (OS) kernel capable of handling output operations, managing time, and overseeing processes.

This project was made in a **Team of 1** as part of the **Operating-System Design Project** course at [Grenoble INP &ndash; Ensimag](https://ensimag.grenoble-inp.fr/).
